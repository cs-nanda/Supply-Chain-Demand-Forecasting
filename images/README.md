# Images

This folder contains the visualizations generated during exploratory data analysis (EDA), model development, and forecasting. These figures complement the analysis presented in the Project Report and Project Presentation.

## Contents

### 📈 eda_trends.png
Visualizes historical demand trends across the dataset, highlighting sales patterns over time.

### 📊 eda_seasonality.png
Illustrates recurring seasonal patterns observed in the retail demand data.

### 📉 mstl_decomposition.png
Shows the decomposition of the time series into trend and multiple seasonal components using the MSTL method.

### 📋 model_metrics.png
Summarizes the forecasting performance of the evaluated models using metrics such as MAE and SMAPE.

### 📌 forecast_comparison.png
Compares the demand forecasts generated by different forecasting models.

### 📅 future_forecast.png
Displays the projected future demand generated by the final selected forecasting model.

### ⚠️ error_distributions.png
Visualizes the distribution of forecasting errors to assess model performance and prediction reliability.

---

These visualizations demonstrate the analytical workflow followed throughout the project, from exploratory data analysis to model evaluation and future demand prediction.
